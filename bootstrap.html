Bootstrap

Descripción: Bootstrap es el framework CSS más popular, desarrollado por Twitter. Permite crear páginas web responsive de manera sencilla usando su sistema de rejilla y múltiples componentes listos.

Instalación

Para instalar Bootstrap en esta página debemos seguir los siguientes pasos:
1. En index.html, dentro de <head> tenemos que añadir:
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css">
2. Justo antes de </body> añadimos el bundle Java Script:
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js">
3. Para verificar que funciona, abrimos index.html y creamos un botón:
<button class="btn btn-primary">Prueba Bootstrap</button>
Si aparece el boton azul, es que funciona.
