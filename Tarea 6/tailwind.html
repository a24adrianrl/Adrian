Tailwind CSS

Descripción: Tailwind CSS utiliza clases utilitarias para construir diseños personalizados. Es muy flexible, moderno y no requiere JavaScript.

Instalación

Para instalar Tailwind en esta página web debemos seguir los siguientes pasos:
1. En <head> de index.html añadimos:
<script src="https://cdn.tailwindcss.com"></script>
2. Verificamos la instalación creando un botón:
<button class="bg-blue-500 text-white px-4 py-2 rounded">Botón Tailwind</button>

Componentes comunes

Botón

<button class="bg-blue-500 text-white px-4 py-2 rounded">Enviar</button>

Formulario

<input class="border p-2 rounded w-full" placeholder="Texto">

Tarjeta

<div class="shadow-lg p-4 bg-white rounded">Contenido</div>

Ventajas

  • Totalmente personalizable.
  • No depende de JS.
  • Ligero y rápido.

Inconvenientes

  • Curva de aprendizaje alta.
  • HTML con muchas clases.
